# practica4-ci-github-actions

# Demo NTFY

Este proyecto demuestra el uso de GitHub Actions para automatizar tareas mediante integración continua.

El repositorio contiene un programa sencillo desarrollado en JavaScript. Cada vez que se realiza un cambio y se envía a la rama principal (`main`), GitHub Actions ejecuta un flujo de trabajo que genera una notificación automática en el canal `ntfy.sh/devops-itla`.

## Tecnologías utilizadas

* JavaScript
* GitHub Actions
* NTFY

## Funcionalidad

* Programa básico en JavaScript.
* Automatización mediante GitHub Actions.
* Envío de alertas al realizar un push en la rama `main`.
